自定义缩放操作点 ​

样式 ​

通过具名插槽handler可以自定义8个缩放操作点的样式

vue
<template>
  <section>
    <FreeDom
      v-model="pos"
      handle
      style="border: 1px solid #ddd;"
    >
      <span>text</span>
      <template #handler="axis">
        <i
          class="custom-handler"
          :class="`custom-handler--${axis}`"
        />
      </template>
    </FreeDom>
  </section>
</template>

<script setup lang="ts">
import { FreeDom } from 'free-dom'
import { ref } from 'vue'

const pos = ref({})
</script>

<style lang="scss">
.custom-handler {
  position: absolute;
  width: 8px;
  height: 8px;
  border-radius: 50%;
  background: #4089ef;
  &--t {
    top: -4px;
    left: 50%;
    margin-left: -4px;
    cursor: ns-resize;
  }
  &--l {
    top: 50%;
    margin-top: -4px;
    left: -4px;
    cursor: ew-resize;
  }
  &--r {
    top: 50%;
    margin-top: -4px;
    right: -4px;
    cursor: ew-resize;
  }
  &--b {
    left: 50%;
    bottom: -4px;
    margin-left: -4px;
    cursor: ns-resize;
  }
  &--lt {
    top: -4px;
    left: -4px;
    cursor: nw-resize;
  }
  &--lb {
    bottom: -4px;
    left: -4px;
    cursor: ne-resize;
  }
  &--rt {
    top: -4px;
    right: -4px;
    cursor: sw-resize;
  }
  &--rb {
    bottom: -4px;
    right: -4px;
    cursor: se-resize;
  }
}
</style>

缩放方向 ​

通过scale可以限制元素的缩放方向

INFO

['l', 't', 'r', 'b', 'lt', 'lb', 'rt', 'rb']八个点分别代表左、上、右、下、左上、左下、右上、右下

vue
<template>
  <section>
    <FreeDom
      :scale="['rb']"
      @update:model-value="pos = $event"
    >
      <pre>{{ pos }}</pre>
    </FreeDom>
  </section>
</template>

<script lang="ts" setup>
import { FreeDom } from 'free-dom'
import { ref } from 'vue'

const pos = ref({})
</script>
